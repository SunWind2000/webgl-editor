# 5. 功能需求

## 5.1 项目与文件管理

| ID | 优先级 | 需求 |
|---|---:|---|
| FR-PROJ-001 | P0 | 创建、打开、另存、关闭项目；支持目录包或单文件索引 + 资产目录。 |
| FR-PROJ-002 | P0 | 支持拖拽、文件对话框、最近文件和操作系统“打开方式”。 |
| FR-PROJ-003 | P0 | 导入任务可暂停/取消；显示阶段、耗时、实体计数、内存和告警。 |
| FR-PROJ-004 | P0 | 保存源文件引用、hash、解析器版本和导入参数；派生数据可失效重建。 |
| FR-PROJ-005 | P1 | 多个模型组合到同一工程坐标系并设置单位、轴向和基点。 |
| FR-PROJ-006 | P1 | 提供项目清理、缓存重建、资源校验、迁移和诊断包导出。 |

## 5.2 查看与渲染

| ID | 优先级 | 需求 |
|---|---:|---|
| FR-REN-001 | P0 | 轨道、平移、缩放、视图立方体、正交/透视、标准视图、适配选择。 |
| FR-REN-002 | P0 | 实体、线框、隐藏线、半透明、法线、分类着色等显示模式。 |
| FR-REN-003 | P0 | 图层、节点、类型和属性可见性；支持 isolate/hide/show all。 |
| FR-REN-004 | P0 | 平面剖切、包围盒、坐标轴、网格、标尺和测量叠加层。 |
| FR-REN-005 | P0 | 按视锥、LOD、屏幕误差和 GPU 预算加载/卸载资源。 |
| FR-REN-006 | P1 | GPU picking、框选/套索、轮廓高亮和海量实例批选。 |
| FR-REN-007 | P1 | 相机书签、视图状态、渲染预设和多视口布局。 |

## 5.3 编辑

| ID | 优先级 | 需求 |
|---|---:|---|
| FR-EDT-001 | P0 | 对象平移、旋转、缩放、复制、删除、重命名和层级调整。 |
| FR-EDT-002 | P0 | 材质颜色、透明度、双面、线宽/点大小和显示样式修改。 |
| FR-EDT-003 | P0 | 所有持久化编辑通过命令执行，支持事务、撤销、重做和失败回滚。 |
| FR-EDT-004 | P1 | 网格顶点/边/面选择与变换、拆分、合并、法线重算和简化。 |
| FR-EDT-005 | P1 | 批量属性修改、选择集保存、图层重组和对象锁定。 |
| FR-EDT-006 | P2 | 精确 B-Rep 拉伸、旋转、布尔、倒角、圆角等受控命令。 |

## 5.4 测量、剖切与标注

| ID | 优先级 | 需求 |
|---|---:|---|
| FR-MEA-001 | P0 | 点点、点线、点面、角度、包围盒、面积和体积测量。 |
| FR-MEA-002 | P0 | 测量结果与实体/拓扑关联，视图变化后保持定位。 |
| FR-SEC-001 | P0 | 支持一个或多个剖切平面，剖切状态可保存。 |
| FR-SEC-002 | P1 | 异步生成精确截面轮廓并允许导出。 |
| FR-ANN-001 | P1 | 文本、引线、标记、问题点和审阅状态。 |

## 5.5 点云

| ID | 优先级 | 需求 |
|---|---:|---|
| FR-PC-001 | P0 | 读取 PCD 常见字段；按 RGB、强度、高程、分类、标签和自定义标量着色。 |
| FR-PC-002 | P1 | 大点云预处理为八叉树/HLOD；相机驱动点预算与渐进细化。 |
| FR-PC-003 | P1 | 框选、裁剪、隐藏、分类过滤和属性统计。 |
| FR-PC-004 | P2 | 配准辅助、地面分割、法向估计和断面分析。 |

## 5.6 导出

| ID | 优先级 | 需求 |
|---|---:|---|
| FR-OUT-001 | P0 | 导出截图、相机书签和简化后的 GLB。 |
| FR-OUT-002 | P1 | 导出选择集、CSV 属性、DXF 2D 实体和项目诊断包。 |
| FR-OUT-003 | P2 | 在明确语义边界下写出 STEP 或更新后的 DWG/DXF。 |

## 5.7 插件与扩展

| ID | 优先级 | 需求 |
|---|---:|---|
| FR-PLG-001 | P1 | 插件可注册菜单、面板、命令、导入器、导出器和分析任务。 |
| FR-PLG-002 | P1 | 插件通过稳定 SDK 访问文档快照、命令总线和只读资源视图。 |
| FR-PLG-003 | P1 | 插件能力声明、版本约束、签名和权限可审计。 |
| FR-PLG-004 | P2 | 支持受控原生插件，但必须独立进程运行。 |
